关于 MiniMessage 编辑器的常见问题

  

          
什么是 MiniMessage 编辑器呢？

          

            

              MiniMessage 编辑器，也叫 Adventure WebUI，是一个专为 MiniMessage 设计的在线编辑器和预览工具。
              它允许您更简单地创建和预览各种各样的 MiniMessage 文本，就好比说聊天消息、物品描述、悬浮文字和服务器列表条目等。
            

          

        

  

          
那什么是 MiniMessage 呢？

          

            

              MiniMessage 是 Adventure 库使用的一种文本格式，用于以紧凑且易于阅读的方式呈现富文本。
              它被各种 Minecraft 插件和模组用于在聊天消息、物品描述、全息图和服务器列表条目中显示格式化文本。
              您可以在官方文档了解更多 MiniMessage 的信息。
            

          

        

  

          
我在哪找 Adventure WebUI / MiniMessage 编辑器的源码？

          

            

               这 Adventure WebUI（也叫 MiniMessage 编辑器）的源码，可以在 GitHub 找到。
            

          

        

  

          
我要怎么用这个 MiniMessage 编辑器/预览器？

          

            

              要使用 MiniMessage 编辑器，只需在屏幕正上方的输入框中输入文本即可。
              您可以使用顶部的按钮选择预览文本的格式，例如聊天、物品描述、悬浮文字或服务器列表条目。
              输出将显示在输入下方。
              您可以使用输入上方的按钮设置文本格式，例如将其设置为粗体、斜体或下划线。
              您还可以添加可点击的 URL、命令和悬停文本。
            

          

        

  

          
我要怎么分享已经生成好的 MiniMessage 文本呢？

          

            

              要分享您生成的 MiniMessage 文本，您可以使用屏幕顶部的分享按钮。
              您可以分享一个您 MiniMessage 文本的短链接，或者是完整链接。
              短链接 6 个月过期，所以如果您想要永久保存您的 MiniMessage 文本，请使用完整链接。
            

          

        

  

          
我要怎么导出我生成好的 MiniMessage 文本到 JSON 文件里面？

          

            

              要导出您已生成的 MiniMessage 文本到 JSON 文件，请点击“导出到JSON”按钮，在屏幕最顶上。
              这将生成您的 MiniMessage 文本的 JSON 表示形式，您可以将其复制并粘贴到您的插件或模组中。
            

          

        

  

          
我要如何在游戏内预览 MiniMessage 文本？

          

            

              要在游戏内预览您的 MiniMessage 文本，请点击“在游戏内测试”按钮，在屏幕最顶上。
              它会将 Minecraft 服务器 IP 复制到您的剪贴板，您可以将其添加到 Minecraft 中的服务器列表中，以便在游戏中预览您的 MiniMessage 文本。
