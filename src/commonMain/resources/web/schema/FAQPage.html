FAQPage (schema.org local copy)

This is a local copy of the schema.org/FAQPage type used for structured data. It is provided to allow the application to reference the type without contacting the external schema.org host.

Definition (summary)

The FAQPage type represents a page containing a collection of questions and answers. It is commonly used to mark up frequently asked questions and their accepted answers for search engines and structured data consumers.

Usage

  • Attach itemscope and itemtype="/schema/FAQPage.html" to the root <html> element (or equivalent) to indicate the page type.
  • Each question should be annotated with itemprop="mainEntity" and itemtype="/schema/Question.html" (or left as text).
  • Each accepted answer should use itemprop="acceptedAnswer" and itemtype="/schema/Answer.html".
