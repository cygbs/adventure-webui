Question (schema.org local copy)

This is a local helper file describing the Question type used with FAQPage structured data.

Definition (summary)

The Question type represents a single question. Use itemprop="name" for the question title and wrap the accepted answer in itemprop="acceptedAnswer".

Example usage:

<div itemscope itemtype="/schema/Question.html" itemprop="mainEntity">
  <h3 itemprop="name">What is this?</h3>
  <div itemscope itemprop="acceptedAnswer" itemtype="/schema/Answer.html">
    <div itemprop="text">This is an example answer.</div>
  </div>
</div>
